Méthodes d’installation

Choisissez votre appareil

💻

Windows — Edge

Ouvrez le site avec Microsoft Edge puis cliquez sur l’icône « Installer cette application » dans la barre d’adresse.

🌐

Windows — Chrome

Ouvrez MPBP440 avec Google Chrome puis cliquez sur l’icône « Installer l’application » dans la barre d’adresse.

📱

Android

Avec Chrome, Edge ou Samsung Internet : menu ⋮ puis « Installer l’application » ou « Ajouter à l’écran d’accueil ».

🍎

iPhone / iPad

Avec Safari : bouton Partager → Ajouter à l’écran d’accueil → Ajouter.
