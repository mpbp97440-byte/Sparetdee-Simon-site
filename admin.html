Mode manuel : images et JSON

  1. Choisis ton image dans l’admin.
  2. L’admin crée un nom propre et un chemin du type assets/covers/mon-titre.png.
  3. Télécharge l’image renommée avec le bouton prévu.
  4. Sur GitHub, upload l’image renommée dans le bon dossier : assets/covers, assets/gallery ou assets/events.
  5. Télécharge ou copie le JSON généré.
  6. Remplace le fichier JSON correspondant dans le dossier data.

Le site actuel reste hébergé sur GitHub Pages : https://mpbp97440-byte.github.io/Sparetdee-Simon-site/
